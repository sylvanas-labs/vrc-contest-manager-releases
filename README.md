The app installs locally and launches as a desktop window.  ## About  VRC Contest Manager is designed to be used by a host during a live VRChat singing contest event, including as a desktop overlay while in VR. Features include contestant management, song assignment, drag-and-drop performance ordering, weighted judging, automatic rankings, and a fullscreen presentation mode.
